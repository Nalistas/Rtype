Detailed Description

This class is used to get all the ressources of a game It takes a std::unique_ptr<rtype::IGame> as parameter It will get all the backgrounds, sprites, musics and client actions of the game It will then transform them into a vector of bytes that can be sent to the clients The vector of bytes will be stored in the _ressources member The getRessourcess method can then be used to get the vector of bytes

R-\/\+Type is a {\bfseries{2D shoot\textquotesingle{}em up}} game inspired by the classic {\itshape R-\/\+Type}, designed and developed by a group of students from EPITECH in their third year. This project incorporates {\bfseries{multiplayer gaming}}, {\bfseries{networking}}, and {\bfseries{real-\/time game mechanics}}. The game features a {\bfseries{client-\/server architecture}} that enables players to connect to a central server, join rooms, and engage in fast-\/paced, multiplayer battles.\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Multiplayer}}\+: Real-\/time multiplayer gameplay using TCP and UDP protocols.
\item {\bfseries{Client-\/\+Server Architecture}}\+: The server handles game state synchronization and player management, while the client provides the interactive gameplay interface.
\item {\bfseries{Graphics}}\+: Developed using the {\bfseries{SFML}} (Simple and Fast Multimedia Library) for rendering and input management.
\item {\bfseries{Network Communication}}\+: The game uses {\bfseries{custom protocols}} (TCP for connections, UDP for real-\/time game communication).
\item {\bfseries{Game Mechanics}}\+: Includes classic elements of the shoot\textquotesingle{}em up genre such as enemy waves, power-\/ups, and challenging levels.
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{Technologies Used}{Technologies Used}}\label{index_autotoc_md3}

\begin{DoxyItemize}
\item {\bfseries{Languages}}\+: C++
\item {\bfseries{Libraries}}\+: SFML for graphics, CMake for build configuration, and custom networking libraries.
\item {\bfseries{Tools}}\+: Git for version control, CMake for project build management.

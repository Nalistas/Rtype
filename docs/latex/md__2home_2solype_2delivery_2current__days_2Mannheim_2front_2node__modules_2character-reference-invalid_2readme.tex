This is a map from the \href{https://html.spec.whatwg.org/multipage/parsing.html\#table-charref-overrides}{\texttt{ HTML spec}} of C1 ASCII/\+Unicode control characters (which are disallowed by HTML) to the characters those code points would have in Windows 1252. For example, U+0080 (Padding Character) maps to {\ttfamily €}, because that’s used for 0x80 in Windows 1252.\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2character-reference-invalid_2readme_autotoc_md1523}{}\doxysection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2character-reference-invalid_2readme_autotoc_md1523}

By default, created descriptor follow characteristics of native ES5 properties, and defines values as\+:

\end{DoxyCode}

\DoxyCodeLine{d("{}ce"{},\ value);\ //\ \{\ configurable:\ true,\ enumerable:\ true,\ writable:\ false\ \}}
\DoxyCodeLine{d("{}e"{},\ value);\ //\ \{\ configurable:\ false,\ enumerable:\ true,\ writable:\ false\ \}}
\DoxyCodeLine{}
\DoxyCodeLine{//\ Same\ way\ for\ get/set:}

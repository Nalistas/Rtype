\chapter{did\+You\+Mean.\+js -\/ A simple Java\+Script matching engine}
\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README}{}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README}\index{didYouMean.js -\/ A simple JavaScript matching engine@{didYouMean.js -\/ A simple JavaScript matching engine}}
\href{https://github.com/dcporter/didyoumean.js}{\texttt{ Available on Git\+Hub}}.

A super-\/simple, highly optimized JS library for matching human-\/quality input to a list of potential matches. You can use it to suggest a misspelled command-\/line utility option to a user, or to offer links to nearby valid URLs on your 404 page. (The examples below are taken from a personal project, my \href{http://dcporter.aws.af.cm/me}{\texttt{ HTML5 business card}}, which uses did\+You\+Mean.\+js to suggest correct URLs from misspelled ones, such as \href{http://dcporter.aws.af.cm/me/instagarm}{\texttt{ dcporter.\+aws.\+af.\+cm/me/instagarm}}.) Uses the \href{https://en.wikipedia.org/wiki/Levenshtein_distance}{\texttt{ Levenshtein distance algorithm}}.

did\+You\+Mean.\+js works in the browser as well as in node.\+js. To install it for use in node\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ install\ didyoumean}

\end{DoxyCode}

\begin{DoxyItemize}
\item str\+: The string input to match.
\item list\+: An array of strings or objects to match against.
\item key (OPTIONAL)\+: If your list array contains objects, you must specify the key which contains the string to match against.
\end{DoxyItemize}

Returns\+: the closest matching string, or null if no strings exceed the threshold.\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1918}{}\doxysection{\texorpdfstring{Options}{Options}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1918}
Options are set on the did\+You\+Mean function object. You may change them at any time.\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1919}{}\doxysubsection{\texorpdfstring{threshold}{threshold}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1919}
By default, the method will only return strings whose edit distance is less than 40\% (0.\+4x) of their length. For example, if a ten-\/letter string is five edits away from its nearest match, the method will return null.

You can control this by setting the "{}threshold"{} value on the did\+You\+Mean function. For example, to set the edit distance threshold to 50\% of the input string\textquotesingle{}s length\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{didYouMean.threshold\ =\ 0.5;}

\end{DoxyCode}


To return the nearest match no matter the threshold, set this value to null.\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1920}{}\doxysubsection{\texorpdfstring{threshold\+Absolute}{threshold\+Absolute}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1920}
This option behaves the same as threshold, but instead takes an integer number of edit steps. For example, if threshold\+Absolute is set to 20 (the default), then the method will only return strings whose edit distance is less than 20. Both options apply.\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1921}{}\doxysubsection{\texorpdfstring{case\+Sensitive}{case\+Sensitive}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1921}
By default, the method will perform case-\/insensitive comparisons. If you wish to force case sensitivity, set the "{}case\+Sensitive"{} value to true\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{didYouMean.caseSensitive\ =\ true;}

\end{DoxyCode}
\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1922}{}\doxysubsection{\texorpdfstring{null\+Result\+Value}{null\+Result\+Value}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1922}
By default, the method will return null if there is no sufficiently close match. You can change this value here.\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1923}{}\doxysubsection{\texorpdfstring{return\+Winning\+Object}{return\+Winning\+Object}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2didyoumean_2README_autotoc_md1923}

\begin{DoxyCode}{0}
\DoxyCodeLine{didYouMean.returnWinningObject\ =\ true;}

\end{DoxyCode}

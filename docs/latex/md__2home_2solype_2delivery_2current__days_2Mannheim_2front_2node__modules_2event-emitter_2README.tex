Pipes all events from {\itshape source} emitter onto {\itshape target} emitter (all events from {\itshape source} emitter will be emitted also on {\itshape target} emitter, but not other way). ~\newline
 Returns {\itshape pipe} object which exposes {\ttfamily pipe.\+close} function. Invoke it to close configured {\itshape pipe}. ~\newline
 It works internally by redefinition of {\ttfamily emit} method, if in your interface this method is referenced differently, provide its name (or symbol) with third argument.\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2event-emitter_2README_autotoc_md2540}{}\doxysubsubsection{\texorpdfstring{unify(emitter1, emitter2) \+\_\+(event-\/emitter/unify)\+\_\+}{unify(emitter1, emitter2) \+\_\+(event-\/emitter/unify)\+\_\+}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2event-emitter_2README_autotoc_md2540}

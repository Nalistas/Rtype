This package is a utility that takes a \href{https://github.com/syntax-tree/hast}{\texttt{ hast}} tree and an \href{https://reactjs.org/blog/2020/09/22/introducing-the-new-jsx-transform.html}{\texttt{ automatic JSX runtime}} and turns the tree into anything you wish.\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2hast-util-to-jsx-runtime_2readme_autotoc_md3089}{}\doxysection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2hast-util-to-jsx-runtime_2readme_autotoc_md3089}
You can use this package when you have a hast syntax tree and want to use it with whatever framework.

This package uses an automatic JSX runtime, which is a sort of lingua franca for frameworks to support JSX.

Notably, automatic runtimes have support for passing extra information in development, and have guaranteed support for fragments.\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2hast-util-to-jsx-runtime_2readme_autotoc_md3090}{}\doxysection{\texorpdfstring{Install}{Install}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2hast-util-to-jsx-runtime_2readme_autotoc_md3090}

\begin{DoxyItemize}
\item Check migration guide to see details for all breaking changes.
\item Breaking\+: "{}unsafe"{} tags {\ttfamily !!js/function}, {\ttfamily !!js/regexp}, {\ttfamily !!js/undefined} are moved to \href{https://github.com/nodeca/js-yaml-js-types}{\texttt{ js-\/yaml-\/js-\/types}} package.
\item Breaking\+: removed {\ttfamily safe\texorpdfstring{$\ast$}{*}} functions. Use {\ttfamily load}, {\ttfamily load\+All}, {\ttfamily dump} instead which are all now safe by default.
\item {\ttfamily yaml.\+DEFAULT\+\_\+\+SAFE\+\_\+\+SCHEMA} and {\ttfamily yaml.\+DEFAULT\+\_\+\+FULL\+\_\+\+SCHEMA} are removed, use {\ttfamily yaml.\+DEFAULT\+\_\+\+SCHEMA} instead.
\item {\ttfamily yaml.\+Schema.\+create(schema, tags)} is removed, use {\ttfamily schema.\+extend(tags)} instead.
\item {\ttfamily !!binary} now always mapped to {\ttfamily Uint8\+Array} on load.
\item Reduced nesting of {\ttfamily /lib} folder.
\item Parse numbers according to YAML 1.\+2 instead of YAML 1.\+1 ({\ttfamily 01234} is now decimal, {\ttfamily 0o1234} is octal, {\ttfamily 1\+:23} is parsed as string instead of base60).
\item {\ttfamily dump()} no longer quotes {\ttfamily \+:}, {\ttfamily \mbox{[}}, {\ttfamily \mbox{]}}, {\ttfamily (}, {\ttfamily )} except when necessary, \#470, \#557.
\item Line and column in exceptions are now formatted as {\ttfamily (X\+:Y)} instead of {\ttfamily at line X, column Y} (also present in compact format), \#332.
\item Code snippet created in exceptions now contains multiple lines with line numbers.
\item {\ttfamily dump()} now serializes {\ttfamily undefined} as {\ttfamily null} in collections and removes keys with {\ttfamily undefined} in mappings, \#571.
\item {\ttfamily dump()} with {\ttfamily skip\+Invalid=true} now serializes invalid items in collections as null.
\item Custom tags starting with {\ttfamily !} are now dumped as {\ttfamily !tag} instead of {\ttfamily !\texorpdfstring{$<$}{<}!tag\texorpdfstring{$>$}{>}}, \#576.
\item Custom tags starting with {\ttfamily tag\+:yaml.\+org,2002\+:} are now shorthanded using {\ttfamily !!}, \#258.

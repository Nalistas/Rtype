This package is a tiny utility that gets the textual content of a node.\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2mdast-util-to-string_2readme_autotoc_md4044}{}\doxysection{\texorpdfstring{When should I use this?}{When should I use this?}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2mdast-util-to-string_2readme_autotoc_md4044}
This utility is useful when you have a node, say a heading, and want to get the text inside it.

This package does not serialize markdown, that’s what \href{https://github.com/syntax-tree/mdast-util-to-markdown}{\texttt{ {\ttfamily mdast-\/util-\/to-\/markdown}}} does.

Get the text content of a node or list of nodes.

Prefers the node’s plain-\/text fields, otherwise serializes its children, and if the given value is an array, serialize the nodes in it.

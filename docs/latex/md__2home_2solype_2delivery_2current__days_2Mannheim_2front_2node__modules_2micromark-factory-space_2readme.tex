Parse spaces and tabs.

There is no {\ttfamily nok} parameter\+:


\begin{DoxyItemize}
\item spaces in markdown are often optional, in which case this factory can be used and {\ttfamily ok} will be switched to whether spaces were found or not
\item one line ending or space can be detected with {\ttfamily markdown\+Space(code)} right before using {\ttfamily factory\+Space}
\end{DoxyItemize}

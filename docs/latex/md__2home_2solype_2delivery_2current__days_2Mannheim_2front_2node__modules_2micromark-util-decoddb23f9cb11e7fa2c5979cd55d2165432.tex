Turn the number (in string form as either hexa-\/ or plain decimal) coming from a numeric character reference into a character.

Sort of like {\ttfamily String.\+from\+Code\+Point(Number.\+parse\+Int(value, base))}, but makes non-\/characters and control characters safe.

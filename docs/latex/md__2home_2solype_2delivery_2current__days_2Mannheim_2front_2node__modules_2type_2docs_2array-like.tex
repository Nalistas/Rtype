Restricted {\itshape array-\/like} confirmation. Returns true for every value that meets following contraints


\begin{DoxyItemize}
\item is an {\itshape object} (or with {\ttfamily allow\+String} option, a {\itshape string})
\item is not a {\itshape function}
\item Exposes {\ttfamily length} that meets \href{array-length.md\#array-lengthcoerce}{\texttt{ {\ttfamily array-\/length}}} constraints
\end{DoxyItemize}

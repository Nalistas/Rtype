Restricted number coercion. Returns number presentation for every value that follows below constraints


\begin{DoxyItemize}
\item is implicitly coercible to number
\item is neither {\ttfamily null} nor {\ttfamily undefined}
\item is not {\ttfamily NaN} and doesn\textquotesingle{}t coerce to {\ttfamily NaN}
\end{DoxyItemize}

For all other values {\ttfamily null} is returned

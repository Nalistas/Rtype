Restricted string coercion. Returns string presentation for every value that follows below constraints


\begin{DoxyItemize}
\item is implicitly coercible to string
\item is neither{\ttfamily null} nor {\ttfamily undefined}
\item its {\ttfamily to\+String} method is not {\ttfamily Object.\+prototype.\+to\+String}
\end{DoxyItemize}

For all other values {\ttfamily null} is returned

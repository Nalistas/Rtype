See \`{}one\`{} ({\ttfamily Function}).\hypertarget{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2zwitch_2readme_autotoc_md7593}{}\doxysubsection{\texorpdfstring{{\ttfamily one(value\mbox{[}, rest…\mbox{]})}}{{\ttfamily one(value\mbox{[}, rest…\mbox{]})}}}\label{md__2home_2solype_2delivery_2current__days_2Mannheim_2front_2node__modules_2zwitch_2readme_autotoc_md7593}
Handle one value. Based on the bound {\ttfamily key}, a respective handler will be called. If {\ttfamily value} is not an object, or doesn’t have a {\ttfamily key} property, the special “invalid” handler will be called. If {\ttfamily value} has an unknown {\ttfamily key}, the special “unknown” handler will be called.

All arguments, and the context object ({\ttfamily this}), are passed through to the handler, and it’s result is returned.

\doxysubparagraph*{{\ttfamily one.\+handlers}}

Map of handlers ({\ttfamily Record\texorpdfstring{$<$}{<}string, Function\texorpdfstring{$>$}{>}}).

\doxysubparagraph*{{\ttfamily one.\+invalid}}

Special \`{}handler\`{} called if a value doesn’t have a {\ttfamily key} property. If not set, {\ttfamily undefined} is returned for invalid values.

\doxysubparagraph*{{\ttfamily one.\+unknown}}

Special \`{}handler\`{} called if a value does not have a matching handler.

\DoxyHorRuler{0}


It\textquotesingle{}s like \href{https://lodash.com}{\texttt{ Lodash}} for dates


\begin{DoxyItemize}
\item It has \href{https://date-fns.org/docs/Getting-Started/}{\texttt{ {\bfseries{200+ functions}} for all occasions}}.
\item {\bfseries{Modular}}\+: Pick what you need. Works with webpack, Browserify, or Rollup and also supports tree-\/shaking.
\item {\bfseries{Native dates}}\+: Uses existing native type. It doesn\textquotesingle{}t extend core objects for safety\textquotesingle{}s sake.
\item {\bfseries{Immutable \& Pure}}\+: Built using pure functions and always returns a new date instance.
\item {\bfseries{Type\+Script}}\+: The library is 100\% Type\+Script with brand-\/new handcrafted types.
\item {\bfseries{I18n}}\+: Dozens of locales. Include only what you need.
\item \href{https://date-fns.org/}{\texttt{ and many more benefits}}
\end{DoxyItemize}

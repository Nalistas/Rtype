Creates a style component with internal {\itshape tracker}.
\begin{DoxyItemize}
\item Adds styles to the browser on the {\bfseries{first}} instance mount.
\item Removes after the {\bfseries{last}} instance unmount.
\item Thus helps you deliver styles you need to the customer, and clean up later.
\item Is not server-\/side rendering compatible!
